2. Assignment Information

Introduction

This assignment helps students apply machine learning theory to solve practical problems on tabular, text, and image data. Students will build data pipelines, train models, and evaluate results to reinforce both theory and hands-on skills.

Goals

  • Apply the traditional machine learning pipeline: preprocessing, feature extraction, model training, and evaluation.
  • Practice with different data types: tabular, text, and image.
  • Analyze and compare model performance using appropriate metrics.
  • Develop programming and scientific reporting skills.

3. Assignment Details

Assignment 1: Tabular Data Assignment: Melbourne Housing Price Analysis & Prediction

  • Dataset: Melbourne Housing Snapshot
  • Notebook: /notebook/251_ML_Assignment1.ipynb   Open in Colab
  • Description: This assignment focuses on analyzing and predicting house prices in Melbourne using machine learning. The project covers data exploration, preprocessing, feature engineering, model training (Linear Regression, SVR, Random Forest, MLP), and interactive experimentation. Key steps include EDA, handling missing/categorical values, feature scaling, PCA, model selection, and performance comparison. Results highlight the impact of location, property features, and model choice on prediction accuracy.
  • How to Run:
    • Google Colab: Click the Colab link above and run all cells for a cloud-based, interactive experience.
    • Local:
      • Go to Assignment_1/ and open a terminal.
      • Install dependencies: pip install -r requirements.txt
      • (Optional) Enable Jupyter widgets: jupyter nbextension enable --py widgetsnbextension
      • Launch Jupyter Notebook: jupyter notebook
      • Open and run notebook/251_ML_Assignment1.ipynb.
  • Key Insights:
    • MLP (Deep Learning) model achieved the highest accuracy (R²), outperforming traditional models.
    • SVR and Random Forest performed best among traditional models.
    • Distance from CBD is a strong negative predictor of price.
    • Dataset quality is high, enabling effective training and evaluation.
