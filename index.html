Assignment 2: Text Data Assignment

  • Report: /report/ML_Assignment2_Report.pdf   Open in Google Drive
  • Dataset: AG News Classification Dataset
  • Notebook: /notebook/251_ML_Assignment2.ipynb   Open in Colab
  • Description:
    • This project focuses on text classification, categorizing news articles into four classes: World, Sports, Business, and Sci/Tech.
    • The core task is to compare the performance of traditional machine learning methods with a modern deep learning approach:
      • Data Preprocessing & EDA: Analyzed class distribution, text length, and word frequencies to understand the dataset.
      • Traditional Models: Used TfidfVectorizer for feature extraction and GridSearchCV to train and evaluate Naive Bayes, Logistic Regression, SVC, and Random Forest.
      • Deep Learning Model: Fine-tuned a pre-trained BERT model to understand the contextual meaning of the news text for classification.
  • Key Insights:
    • The fine-tuned BERT (Deep Learning) model achieved the highest test accuracy (91.3%), significantly outperforming traditional methods.
    • SVC (Support Vector Classifier) was the best-performing traditional model, providing a strong baseline with 87.1% accuracy.
    • This comparison highlights the superior ability of transformer-based models like BERT to understand language context for classification tasks.
    • While computationally more expensive, the deep learning approach offers a clear advantage in performance for this text-based problem.

Assignment 3: Image Classification Assignment

  • Report:Open in Google Drive
  • Dataset: Mammals Image Classification Dataset
  • Notebook: /notebook/251_ML_Assignment3.ipynb   Open in Colab
  • Description:
    • This project focuses on image classification, categorizing articles into 45 mammal species.
    • The core task is to compare traditional machine learning (with engineered features) against modern deep learning (with transfer learning).
      • Data Preprocessing & EDA: Analyzed class distribution (mild imbalance), image brightness, and performed stratified 60/20/20 splitting.
      • Traditional Models: Extracted features using HOG+PCA, VGG-16 (features), and EfficientNet-B0 (features), then trained SVM, Logistic Regression, and Random Forest.
      • Deep Learning Model: Fine-tuned a pre-trained ResNet-50 model end-to-end.
  • Key Insights:
    • The EfficientNet-B0 (feature extractor) + Logistic Regression combination achieved the highest test accuracy (93.4%).
    • The HOG+PCA pipeline (traditional feature engineering) failed completely, with <15% accuracy, demonstrating the necessity of deep learning features.
    • SVC, Random Forest, and Logistic Regression all performed nearly identically (>87%) *once* given high-quality features from VGG, ResNet, or EfficientNet.
    • This comparison highlights that the "battle" was won at the feature extraction stage, not the final classification stage.

Assignment Extra: Bayesian Network Implementation

Goal: This module implements a Bayesian Network from scratch to model uncertain relationships in a student performance domain. It features a custom Directed Acyclic Graph (DAG) structure, conditional probability tables (CPTs), and both exact and approximate inference algorithms.

Source Code: /Assignment_Extra/bayesian.ipynb   Local Only

1. The Student Performance Model

We modeled the causal relationships between student characteristics and their academic outcomes.

Difficulty (D) Intelligence (I) \ / | \ / | v v v Grade (G) SAT (S) | v Letter (L)
  • Variables: Difficulty, Intelligence, Grade (A, B, C), SAT, Letter.
  • Task: Calculate posterior probabilities of hidden variables given observed evidence.

3. Performance Analysis

Method Accuracy Theoretical Speed Use Case
Exact (Enumeration) 100% (Baseline) Slow Small networks, critical precision.
Approx (Sampling) ~98-99% (with N=10k) Fast Large, complex networks.

In the experiments, Rejection Sampling with 10,000 samples achieved results within 1.5% margin of error compared to the exact calculation. The theoretical speed advantage of sampling methods is not fully realized in this small network due to overhead, but becomes significant in larger networks.
